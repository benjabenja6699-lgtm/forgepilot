# Workflow Launcher

Tabbed launcher for:

- `Instaladores`
- `Configuraciones`
- Windows and Linux sub-tabs
- `Logs`
- `Settings`

## Run

```powershell
python .\workflow_launcher.py
```

## Build exe

```powershell
.\build.ps1
```

The exe lands in:

```text
tools/workflow-launcher/dist/AIWorkflowLauncher.exe
```
