# Workflow Launcher

Windows launcher for:

- Graphify install
- Caveman skill install
- repo AI rules kit setup
- AGENTS.md install
- git hook install
- Graphify map build

## Run

```powershell
python .\workflow_launcher.py
```

## Build exe

```powershell
.\build.ps1
```

The exe lands in:

```text
tools/workflow-launcher/dist/AIWorkflowLauncher.exe
```
